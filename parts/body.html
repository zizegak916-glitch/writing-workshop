隐私条款与数据使用说明

📦 本地存储

所有数据（项目、文本、设置、API配置）仅存储在浏览器 localStorage 和 IndexedDB 中，不会上传到任何服务器。

🍪 Cookie 使用

使用 localStorage 保存用户偏好和登录状态，不使用第三方跟踪 Cookie。

🔑 API 调用

AI 功能通过用户自行配置的 API 密钥调用，密钥仅存储在本地浏览器中。

🛡️ 数据安全

可随时通过「清除数据」功能删除所有本地存储的数据。您的数据完全由您掌控。

🌐 第三方服务

使用 Google Fonts 等 CDN 资源，可能涉及 IP 地址等基本网络信息。

📬 联系方式

如有疑问，请通过 GitHub Issues 联系我们。

本网站使用 Cookie 和本地存储来提升您的使用体验。继续使用即表示您同意我们的隐私条款。
