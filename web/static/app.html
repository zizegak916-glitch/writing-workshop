隐私条款与数据使用说明

📦 本地存储

所有数据（项目、文本、设置、API配置）仅存储在浏览器 localStorage 和 IndexedDB 中，不会上传到任何服务器。

🍪 Cookie 使用

使用 localStorage 保存用户偏好和登录状态，不使用第三方跟踪 Cookie。

🔑 API 调用

AI 功能通过用户自行配置的 API 密钥调用，密钥仅存储在本地浏览器中。

🛡️ 数据安全

可随时通过「清除数据」功能删除所有本地存储的数据。您的数据完全由您掌控。

🌐 第三方服务

静态资源由 ainovel-cli 单二进制本地提供，不加载外部 CDN 字体或脚本。

📬 联系方式

如有疑问，请通过 GitHub Issues 或 Telegram 联系我们。

📖 使用指南

完整的使用文档请访问：使用文档

🚀 快速开始

  1. 打开应用，首次使用请设置密码（至少4位）
  2. 进入工坊后，点击顶栏「＋」创建新项目
  3. 点击「⚙」配置 API Key（选服务商 → 填 Key → 测试 → 保存）
  4. 开始写作！AI 功能会自动辅助你

🔧 API 配置

支持 11+ 服务商：

  • DeepSeek（推荐，性价比高）— deepseek.com
  • 通义千问 — tongyi.aliyun.com
  • 智谱 GLM — open.bigmodel.cn
  • 小米 MiMo — api.xiaomi.com
  • OpenAI — platform.openai.com
  • Claude — console.anthropic.com
  • Kimi/Moonshot — platform.moonshot.cn
  • Gemini — aistudio.google.com
  • Grok — console.x.ai
  • OpenRouter — openrouter.ai
  • SiliconFlow — siliconflow.cn
  • 自定义（任意 OpenAI 兼容 API）

✍️ 编辑器功能

  • AI 快捷工具栏（编辑器上方）：纠错、自动标题、实时灵感、资料搜索、降AI率、查AI
  • 工具栏：专注模式、预览、加粗/斜体、字体大小、导出
  • 自动保存：开启后实时保存，关闭后手动 Ctrl+S

📱 手机端底部导航

  • 编辑 — 写作主界面
  • 大纲 — 故事大纲管理
  • 章节 — 章节列表与切换
  • 人物 — 人物档案卡片
  • AI — AI 助手面板（30+ 模式、多AI对比）
  • 隐私 — 查看隐私条款与使用指南

🤖 AI 面板功能

  • 助手 Tab：选择 AI 模式（续写/润色/扩写等30+），调整创意度和输出长度，点「开始生成」
  • 对比 Tab：3 个槽位并行对比不同 AI 服务商的结果
  • 记忆 Tab：添加记忆条目，AI 生成时自动注入上下文
  • 历史 Tab：查看所有 AI 操作记录

👤 个人中心

点击顶栏「👤」查看项目统计、修改密码、锁屏。

🎨 主题切换

设置 → 主题/Theme → 深色/浅色模式。支持跟随系统。

📊 数据安全

  • 所有数据存在浏览器本地（IndexedDB）
  • 密码用 SHA-256 加密存储
  • 不上传任何数据到服务器
  • 支持项目导出 JSON 备份
  • 支持导入 .json / .txt / .md / .docx 文件

❓ 常见问题

Q: 刷新页面要重新登录吗？
A: 不需要，刷新直接进入。点「👤」里的「🔒 锁屏」才会锁。

Q: 数据会丢吗？
A: 数据存在浏览器本地。清除浏览器数据会丢失，建议定期导出备份。

Q: AI 功能需要付费吗？
A: 需要 AI 服务商的 API Key，大部分有免费额度。

Q: 手机上能用吗？
A: 完全适配手机，底部导航栏切换功能。
