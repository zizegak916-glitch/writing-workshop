使用文档

AI写作工坊现在由 ainovel-cli 的 Go 后端提供 API，并通过 GitHub Pages 发布静态前端。

打开写作工坊

快速开始

go build -o ainovel-cli ./cmd/ainovel-cli
./ainovel-cli serve --port 8080

本地服务启动后访问 http://127.0.0.1:8080/app.html。Pages 静态版可浏览界面，真实 AI 调用需要本地 Go 后端或已部署的 API 服务。

核心入口

写作工坊

app.html，用于项目、章节、角色、AI 辅助写作。

管理后台

admin.html，用于配置 provider、model、API key 和规则包。

API

后端端点以 /api/ 开头，主要包括 /api/ai、/api/config、/api/rules。

API Key 配置

推荐在管理后台保存 provider 配置，或使用环境变量，例如 AINOVEL_OPENROUTER_API_KEY、OPENROUTER_API_KEY、OPENAI_API_KEY。
