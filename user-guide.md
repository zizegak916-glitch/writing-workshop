# AI 写作工坊 使用指南

## 打开方式
手机浏览器打开：https://zizegak916-glitch.github.io/writing-workshop/

## 首次使用
1. 打开链接，设置密码（至少4位）
2. 进入工坊后，点顶栏「＋」创建项目
3. 点「⚙」配置API（填API Key，选服务商）

## API 配置
支持的服务商：
- DeepSeek（推荐，便宜）
- 通义千问
- 智谱GLM
- 小米MiMo
- OpenAI
- Claude
- Kimi/Moonshot
- OpenRouter
- SiliconFlow
- 自定义

## 编辑器功能
- **AI快捷工具栏**（编辑器上方）：
  - ✓ 纠错 — 检查全文错别字和语病
  - ¶ 自动标题 — 根据内容生成标题
  - ◇ 实时灵感 — 根据上下文提供创作建议
  - ⊕ 资料搜索 — 搜索相关背景知识
  - ▷ 降AI率 — 重写文字降低AI检测率
  - ⊕ 查AI — 分析文字的AI特征（仅供参考）

- **工具栏**：专注模式、预览、加粗/斜体、字体大小、导出

## 底部导航（手机端）
- ✎ 编辑 — 写作主界面
- ☰ 大纲 — 故事大纲管理
- ▤ 章节 — 章节列表
- ◉ 人物 — 人物档案
- ⚡ AI — AI助手面板（30种模式）

## AI面板
- **助手Tab**：选择AI模式，输入要求，点「开始生成」
- **对比Tab**：3个槽位并行对比不同AI
- **历史Tab**：查看AI操作记录

## 个人中心
点顶栏「👤」按钮：
- 查看项目统计（字数/大纲/章节/人物）
- 修改密码
- 锁屏

## 主题切换
点顶栏「🌙/☀️」按钮切换暗色/亮色主题

## 数据安全
- 所有数据存在浏览器本地（IndexedDB）
- 密码用SHA-256加密
- 不上传任何数据到服务器
- 支持项目导出JSON备份

## 常见问题
Q: 刷新页面要重新登录吗？
A: 不需要，刷新直接进入。点「👤」里的「🔒 锁屏」才会锁。

Q: 数据会丢吗？
A: 不会，数据存在浏览器本地。但清除浏览器数据会丢失，建议定期导出备份。

Q: AI功能需要付费吗？
A: 需要自己有AI服务商的API Key，大部分服务商有免费额度。

Q: 手机上能用吗？
A: 完全适配手机，底部导航栏切换功能。
